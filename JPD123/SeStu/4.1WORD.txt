*】　文法　：
１。 Hỏi đất nước của bạn là gì?　
　お　にく　は　どちら　ですか？
⇒Tên đất nước+　です。
たとえば：たい｜タイ：nước Thái
	べとなむ｜ベトナム：nước Việt Nam
	にほん｜ニホン｜日本　：nước Nhật
	かんこく｜カンコク｜韓国：nước Hàn
	ちゅうごく｜チュゴク｜中国：nước Trung Quốc
	あめりか｜アメリカ｜：nước Mỹ
	いたりあ｜イタリア：nước ý
	おすとらりあ｜オーストラリア：nước Úc
	ろしあ｜ロシア：nước Nga
	ぶらじる｜ブラジル：nước Brazil　
２。Hỏi ở đâu của đất nước ?
　A　+　のどこ　+　ですか？
⇒　Ｂ＋　です。
たとえば：１】　タイ　の　どこ　ですか？
⇒　アュタヤ　です。
２】　ベトナム　の　どこ　ですか？
⇒　ハノイ　です。
３。Hỏi ở đâu của địa điểm trong đất nước ?
　A　+　は　+　どこ　+　ですか？
⇒A　+　は　+　Ｂ　+　の　+　ví trí（ひがし／にし／きた／みなみ／まんなか） + です。
たとえば：１】　アュタヤ　は　どこ　ですか？
⇒アュタヤ　は　タイ　の　まんなか　です。
２】　ハノイ　は　どこ　ですか？
⇒”ハノイ　は”　ベトナム　の　きた　です。

例：ほっかいどう　は　日本　の　きた　です。
１）ながの　は　日本　の　まんなか　です。
２）おきなわ　は　日本　の　みなみ　です。
３）パース　は　オーストラリア　の　にし　です。
４）シドニー　は　オーストラリア　の　ひがし　です。

ホーチミン　は　ベトナム　の　みなみ　です。
とうきょう　は　ほんしゅう　の　ひがし　です。

４。Hỏi đi từ A đến B bằng phương tiện gì?　
A　+　から　+　B　+　まで　なんで　いきますか？
A　+　から　+　B　+　まで　何で　行きますか？
⇒Phương tiện + ”で”　+　行きます。
例：
A：うち　から　かいしゃ　まで　何で　行きますか？
B：⇒じてんしゃ　で　行きます。　🚴
　⇒くるま　で　行きます。　🚗
　⇒あるいて　行きます。　（đi bộ)

５。Từ đâu đến đâu mất bao lâu ?　
A　+　から　+　B　+　まで　どのくらい　ですか？
⇒A　+　から　+　B　+　まで　+　phương tiên + "で"　
			+　thời gian + （じかん／ふん｜ぷん）+　くらい（khoảng）+　です。　
Đếm số:Số + じ：よじ，しちじ，くじ：4，7，9
Đếm phút:Số +　ふん（ぷん）
ふん：2，5，7，9

ぷん：1，3，4，6，8，10
いっぷん，さんぷん，よんぷん，ろっぷん，はっぷん，じゅっぷん
Đếm tiếng:Số + じかん：4，7，9　　　　　　
例：

ベトナム　から　日本　まで　6　時間　くらい　です。
ハノイ　から　ホーチミン　まで　２時間　くらい　です。

うち　から　だいがく　まで　あるいて　15　分　です。
ハノイ　から　フエ　まで　でんしゃで　12　じかんくらい　です。
